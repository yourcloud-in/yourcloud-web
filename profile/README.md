## 🌍 What is GravRel?

GravRel is India's first **solar-powered**, **DPDP Act 2023 compliant** cloud infrastructure platform.
Built for startups, researchers, and enterprises in Tier 2 India.

- 🛡️ **Data never leaves India** — DPDP compliant from day one
- ☀️ **Zero carbon emissions** — 100% solar powered, zero grid electricity
- 💧 **Zero water waste** — air-cooled only, no liquid cooling ever
- 💰 **60% cheaper than AWS Mumbai** — starting at ₹199/month
- 📞 **CEO personally reachable** — ceo@gravrel.com
